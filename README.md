## Windows 10 OS
#Prerequisites

Download and install Vagrant from https://www.vagrantup.com/downloads

## open cmd and run below commands 

mkdir %HOMEPATH%\bina

git clone https://github.com/sysadmingroup/rhcsa.git .
vagrant up
